About

Diese Website ist im Rahmen eines Softwareprojektes entstanden, welches im vierten Semester an der ZHAW durchgeführt wird. Die Themenwahl beschränkte sich auf die Implementierung von gängigen Algorithmen oder numerischen Verfahren zur Lösung eines Problems.

Wir haben uns deshalb für die Implementierung eines neuronalen Netzes entschieden, welches gezeichnete Zahlen erkennen soll. Diese Webseite stellt dabei das Frontend dar. Das eigentlche neuronale Netz ist auf einem Heroku-Server zu finden. Das API dazu kann auf www.github.com/kressi/neural_net gefunden werden.

Wie man unschwer erkennen kann, funktionier die Zeichenerkennung noch nicht einwandfrei. Dies liegt vor allem an den mangelden Recourcen, welche zum Trainieren des Netzes benötigt werden. Je mehr ein Netz trainiert wird, desto bessere Daten kann es zurückliefern.
